Specifiche dei requisiti

1. Requisiti Utenti
  1.1 nome: String
  1.2 data_reg: Data
  1.3 video pubblicati (v. req. 2)
  1.4 video visualizzati con quando sono stati visualizzati e il numero delle volte e una valutazione (da 0 a 5)
  1.5 commentare un video,

2. Requisiti Video
  2.1 titolo: String
  2.2 durata_sec: int > 0
  2.3 descrizione: String
  2.4 nome_file: String
  2.5 categoria (unica) di cui fa parte (v. req. 3)
  2.6 tag alemeno uno (v. req. 4)
  2.7 un video puo essere pubblicato in risposta ad un altro video
      (non puo essere un video pubblicato dallo stesso utente)
  2.8 numero visualizzazioni (operazione)

3. Requisiti Categoria 
  3.1 nome: String

4. Requisiti Tag
  4.1 nome: String
